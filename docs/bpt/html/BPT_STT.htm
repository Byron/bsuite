The Soft Transformation Tool


The Soft Transformation Tool (->SST) essentially behaves like the native Maya Move Tool, except that you can define an influence radius to affect all the components inside it.
Additionally you can push your selection along the normal, or scale it.
Please Note: It will only work on polygonal components (vertices, edges, faces). It works on vertices internally, so any other component type will be converted implicitly.
Additionally you should know about some Undo/Redo Issues that will be the matter later.

 Starting the Tool

1. Via the BPT Menu

-> Click on ByronsPolyTools 1.0 -> Transformation -> Soft Transformation Tool

OR

2. Manually

-> Execute the following command: setSoftTool


 Tool Options

The Use Last Distance parameter allows you to determine whether the tool should save the distance you used with the last selection and reuse it in the next one.
This allows you "tweak" you mesh more quickly because you don't have to drag the distance with every selection change.

 Using the Tool
  1. Enter the Tool as described above.
  2. Select at least one polygonal component
  3. Adjust your soft selection radius by left-clicking directly on one of the components (i.E. the center of faces), and drag the pointer to the right to increase the falloff distance, or drag it to the left to decrease it. Release the mouse button if you are done.

    You can also use the HotKeys: [+] to increase distance and [-] to decrease distance.

    3a.(Optional) Adjust the MaskSize by clicking directly on one of the components with the left and middle mouse button. Drag the pointer to the right to increase the maskSize, or drag it to the left to decrease it.

    You can also use the hotkeys: [Ctrl] + [+] to increase maskSize, and [Ctrl] + [-] to decrese maskSize

 Changing Falloff Type
The STTool offers 4 different falloff types: Linear, Smooth, Spike and Dome (in this order).
You should use the state manipulator to change the falloff.

Please click here for information on the different falloff types.

 Masking

As with the ByronsPolyToolsNode, you can use the maskSize attribute to limit your influence field with a mask which grows out of your component selection.

To use the masking, you can do one of the following:

- directly click with the left and middle mouse button on a selected component and drag it to the right to increase or to the left to decrease the maskSize.

OR

- Open the AttributeEditor ([Ctrl] + [a]), select the softTransformationNode which is directly behind the meshes shapeNode and adjust the Mask Size parameter.

If this parameter is zero (default) the influence field won't be limited.


 Tool Hotkeys

In the moment you load the Plugin the first time, some HotKeys will be installed which will make work more convenient.

Now you can press ...

[+]: To increase the distance.

[-]: To decrease the distance.

[Ctrl] + [+]: To increase the maskSize.

[Ctrl] + [-]: To decrease the maskSize.

If you want to create, for what reason ever, these hotkeys manually, please refer to the following section.


 Manually Customizing the Tool

The Hotkeys can be made also manually. If you want to do that, you can do the following:

Mapping "IncreseDistance" to the [+] Key:

  1. Open the Hotkey Editor (Windows -> Settings/Preferences->Hotkeys ... ).
  2. Choose the "User" Tab on the left side.
  3. Click on "New" and enter a name in the "Name" Field. I will use "STTincreaseDistance" in this Example.
  4. Click on "Edit"
  5. In the "Command" Field, copy-paste the following lines:
    if(`currentCtx` == "softTransformationTool1")
{
	eval("softTransformationTool -e -rdc 32 softTransformationTool1");
}
else	//use standardBehavior if not inside STTool
	manipOptions -r -s 2.0
    The figure next to the "-rdc" ("relativeDistanceChange") Flag will be the increment you use to change the distance. The default value will be a trade-off between precision und imprecision, but you can use every figure you like.
  6. Click "Accept".
  7. In the "Assign New Hotkey" Section, enter a "+" into the "Key" Field and make sure that the "Modifier" checkboxes are cleared.
  8. Hit "Assign" and confirm the following Question.

Mapping "DecreseDistance" to the [-] Key:

  • Repeat steps 3 to 8, but
  • In Step 3, fill "STTdecreaseDistance" in the "Name" field.
  • In Step 5, use the following lines: 
    if(`currentCtx` == "softTransformationTool1")
{
	eval("softTransformationTool -e -rdc -32 softTransformationTool1");
}
else	//use standardBehavior if not inside STTool
	manipOptions -r -s 0.5
    If you use a custom figure, simply negate it.
  • In Step 7, enter a "-" into the "Key" Field and make sure that the "Modifier" checkboxes are cleared.

Mapping "IncreseMaskSize" to the [Ctrl] + [+] Key:

  • Repeat steps 3 to 8, but
  • In Step 3, fill "STTincreaseMaskSize" in the "Name" field.
  • In Step 5, use the following lines: 
    if(`currentCtx` == "softTransformationTool1")
{
	eval("softTransformationTool -e -rmc 1 softTransformationTool1");
}
else	//Insert a command of your choice which will apply 
	//if STTool is not active
	/*CommandName*/;

    • You could also enter a custom figure behind the "-rmc" ("relativeMaskChange") Flag.

    Behind the else statement you could enter custom comman which will be applied when the STTool is not active. To do that, simply replace "/*commandName*/ with your command.

  • In Step 7, enter a "+" into the "Key" Field and make sure that the "Modifier" checkbox "Ctrl" is selected.

Mapping "DecreseMaskSize" to the [Ctrl] + [-] Key:

  • Repeat steps 3 to 8, but
  • In Step 3, fill "STTdecreaseMaskSize" in the "Name" field.
  • In Step 5, use the following lines: 
    if(`currentCtx` == "softTransformationTool1")
{
	eval("softTransformationTool -e -rmc -1 softTransformationTool1");
}
else	//Insert a command of your choice which will apply 
	//if STTool is not active
	/*CommandName*/;

    • You you used a custom figure, it should be negated.

  • In Step 7, enter a "-" into the "Key" Field and make sure that the "Modifier" checkbox "Ctrl" is selected.

 


 The Soft Transformation Node

The Soft Transformation Node can be considered the engine of the Soft Transformation Tool, therefore it will do all the work. It will only be temporarily in your node's history.
The Manipulator actually accesses some of the nodes parameters. You can change the parameters directly and enter numerical values if you show up the Attribute Editor and choose the softTransformationNode1 tab while inside of the Soft Transformation Tool (with components selected).
The only useful parameter that can only be accessed with the AE is the normalRelative parameter. It determines whether the pushing/alongNormal translation will be scaled according to the faces size ("normalRelative == false") or if they should all translate the same way ("normalRelative == true").

Please Note: To prevent you from clicking in the channelBox while the STTool is active (which would cause a Maya Internal Error), it will be deactivated during that time.

 Undo/ Redo the Tool


Because the UR mechanism also registers every parameter change on the STNode which doesn't necessarily exist anymore then, there may be several "empty undo's" on the queue.

It is not recommended to hit undo until your selection changes while the STT is acitve. You should favour to leave the tools first and then undo.
