Selection


The Selection Commands you learn about here will help you to quickly select the components you want without having to single-pick them.

 Smart Edge Loop/Loop


Edge Ring and Edge Loop allows you to select a ring or loop of edges just by picking a component.

ByronsPolyTools -Flag ...
LongName ShortName Description
-edgeRing -er - Selects the edgeRing corresponding to your componentSelection
-edgeLoop -el - Selects the edgeLoop corresponding to your componentSelection
LongName ShortName --Additional Flags--
-maxEdgeCount -mec - Limit the ring/loop to to the specified count
Example: ByronsPolyTools -er -mec 10
-autoSwitchComponentType -ast - automatically switch the component mask to edges
Example: ByronsPolyTools -er -ast -mec 10

 Philosophy

The main idea behind the edge Ring and edge Loop command is based on two things: First, you are not limited to a specific component type. This means, you can also select faces and/or vertices and select their corresponding ring or loop paths. On the other hand, you will be able to block a path by selecting another component (typically edges) on it. The created path will be between these stop Edges. If you use this feature on a ring/loop cycle, it will select the shortest path.

You can also select pathes on boundaries and use the stop feature on them.
