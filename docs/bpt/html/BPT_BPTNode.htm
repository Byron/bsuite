The ByronsPolyTools Node

This Chapter introduces you to the ByronsPolyToolsNode (->BPTN). It is the core of ByronsPolyTools, and therefore it is vital to know about its parameters and abilities.
Every command that creates a BPTN will be able to use all of its parameters in the same way. SlideRelative,NormalRelative,Side,AlongNormal and Slide will be explained in detail in the following chapters beginning with Smart Split.
That is why this chapter will concentrate on the parameters that control the built in Soft Selection


 The SoftSelection

SoftSelections can be controlled very precisely, meaning that you can individually adjust the distances for "alongNormal" and "slide" with the alongNormalTweakDistance and slideTweakDistance parameters, which will determine the size of each influence field.
The maskSize parameter will grow a mask out of your original selection, restricting the area which will be influenced.
The Falloff Visualization parameter allows you to control whether you want to view the weights (which are shown using Vertex Colors) or not.
The Mesh Visualization parameter will show the mesh without the deformation imposed by the BPTNode if set to on.
Additionally you can choose between 4 different falloff type presets and a custom curve to control the way the falloff dimishes.

 Custom (not in STTool)

To use an own custom curve, make sure that a ByronsPolyToolsNode is selected and at least one falloff type is set to custom.

Open the graph Editor. There will be two AnimationCurves, named customANFalloff and customSlFalloff which will change the falloff for the alongNormalFalloffType or the slideFalloffType respectively.
Now you can change the curve freely, but it is recommended to keep the keys within their -0.5 <= x <= 0.5 boundary.

It may be necessary to change the the "alongNormal" or "slide" attributes to force an update in the viewports.

Please Note: You should only change the curve when the falloffType is set to custom, because the changes won't have any effect otherwise.
