Description

This node will draw the vertices of mesh as dots with custom color (its basically a slightly changed devkit example).

This node can be considered some sort of bonus, just because it is not used by BPT.

It is included since it can be useful to visualize goals of softbodies.

Usage:

Connect the mesh's "outMesh" attribute to the visualizeMeshNode's "inputMesh" Attribute.
