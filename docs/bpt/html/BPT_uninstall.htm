Uninstalling ByronsPolyTools

To uninstall BPT you need to execute the uninstall procedure which will do the following:

  • Delete the option variables used to keep the preferences
  • Delete the Soft Transformation Tools hotkeys and restore the keys default behavior

To achieve this, do the following:

  1. Execute the following command: 
    BPT_uninstall
  2. Unload the Plugin
    WARNING: In Maya 5.01 a crash will occour when you try to unload the plugin. To workaround this case you should simply don't unload the plug. Autoload has been torn off be the above script. 
  3. Delete Plugin's files.

If you want to install it again, just load the Plugin (if you did not delete the Plugin's files).
