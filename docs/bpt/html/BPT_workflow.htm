Workflow


This section is intended to help you using the tools in the fastest way possible, and to make Maya a competitor to every modeler out there in terms of handling and speed. I will write about it in general and give a detailed example based on my own modeling layout.
To have one of the strongest polymodelling toolsets is the one thing, and the other one will be to use the commands within a fraction of a second - its modeling what counts, so there is not time to click on buttons.

 The BPT Menu

The menu only serves the purpose to give you a fast introduction into the tools, to show to you the commands and the modifiers/flags which change the behavior of the commands and to manage your preferences.
It is definitely not meant to be used in everyday modeling because its simply to time consuming to move your pointer to the respective buttons.
In general, it was best when it would take no time to invoke a command, so you could completely care about modeling.

 Using Hotkeys

Hotkeys are vital for a good workflow, because they allow to invoke commands extraordinarily quick.

Before we begin to think about creating our own ones, we should always try to keep the following guides in mind:

1. Minimize the amount of special purpose hotkeys (i.e. those for modeling) to an absolute minimum - keys should be pressed without having to intentionally think about it.

2. All hotkeys necessary should be located in the same quadrant of the keyboard to allow the palm to be static (this quadrant is normally on the left side of the keyboard). The longer the ways of your fingers to the key, the more you will get out of your creative process because you have to wait for an action (which perhaps is already done in your head).

3. Single hotkeys should be assigned the commands you use most often, and complex ones (i.e. Ctrl + Alt + Shift + a) those whose usage is seldom.
In general you should try to avoid complex hotkeys if possible because it takes longer to press them.

 Hotkeys - Selection Masks

Regarding the above guides, the first thing to change have to be the hotkeys for the polygon component selection masks.
With default preferences, you would have to move your whole hand to the F8 to F12 keys, which can be real bothering.
I chose to use the keys 1, 2 and 3 to be mapped to the component masks for vertices, edges and faces respectively, while shift + 1 will change back to object mode.
Normally these keys are meant to change the display quality of NURBS and Subdivision Surfaces, but as a polymodeler, these keys are in use quite seldom.

The following scripts would have to be mapped to the corresponding keys:
(to learn more about assigning hotkeys, please refer to the Maya Documentation and execute the following: showHelp HotkeyEditora )

[ 1 ] : 		setSelectMode components Components;
			selectType -smp 1 -sme 0 -smf 0 -smu 0 -pv 1 -pe 0 -pf 0 -puv 0; 
			select -cl;


[ 2 ] :			setSelectMode components Components; 
			selectType -smp 0 -sme 1 -smf 0 -smu 0 -pv 0 -pe 1 -pf 0 -puv 0; 
			select -cl;


[ 3 ] :			setSelectMode components Components; 
			selectType -smp 0 -sme 0 -smf 1 -smu 0 -pv 0 -pe 0 -pf 1 -puv 0; 
			select -cl;


[SHIFT] + [ 1 ] :	setSelectMode("objects", "Objects");



 Hotkeys - Component Type Conversions

It is a fact that is is very usefull to be able to quickly convert between selection types. For example, you could select an edge Loop, convert it to faces and then invoke a smartMoveComponent which would move this selection then as if it was a solid Chamfer.

In my layout, I use the following scheme with the respective MEL Command:
[Alt] + [ 1 ] :				PolySelectConvert 3;
					setSelectMode components Components;
					selectType -smp 1 -sme 0 -smf 0 -smu 0 -pv 1 -pe 0 -pf 0 -puv 0;

[Alt] + [ 2 ] :				PolySelectConvert 2;
					setSelectMode components Components; 
					selectType -smp 0 -sme 1 -smf 0 -smu 0 -pv 0 -pe 1 -pf 0 -puv 0;

[Alt] + [ 3 ] :				PolySelectConvert 1;
					setSelectMode components Components; 
					selectType -smp 0 -sme 0 -smf 1 -smu 0 -pv 0 -pe 0 -pf 1 -puv 0;

The hotkeys above will convert the component selection into vertices([alt] + [1]), edges ([alt] + [2]) and faces ([alt] + [3]).
If you like wings3D, you could also use these commands and put it onto your standard component Mask hotkeys (for example 1,2 and 3) to automatically convert your selection when you change component masks.

 Hotkeys - Delete & Move Pivot

If we take into account the guides for hotkeys, then the Delete and InsertKeys are definitely too far away. I personally prefer to use the following layout which allows quick access to these functions:

[SHIFT] + [^] : 		delete;

[^]:				EnterEditMode;

Please Note: The [^]-Key does only work that way when the "EN" Keyboard Layout is active.

By the way, it is very usefull to have the English keyboard layout enabled for the appplication because the undo/redo shortcuts are much closer then (compared to the german layout). As an alternative to switching the layout you could manually change the hotkeys for undo and redo which would have the advantage that you don't have to care about the current layout.

Often it is necessary to center the pivot when you just created or duplicated an object so there could also be a hotkey for that. I hope, in the next Maya version there will be an option like "AutoCenterPivot" so that the user is not forced to care about that.

 Hotkeys - Assigning BPT Commands

There are two types of ByronsPolyTools Hotkeys: Static and dynamic ones.

Static Hotkeys consist of a simple lowLevel Command, for example:

ByronsPolyTools -sp -sl 0.5 -relative -snl -avf -bp

These will always invoke the same action, no matter what options are set in the preferences or in the subMenus.
To get such a command, you can use the Preferences -- Show Command -- CommandName (Explanation) options in the BPT Menu.
Additionally they will be printed into the script Editor when you invoke a command.


Dynamic Hotkeys will be mapped to procedures which are located in the ByronsPolyToolsMenu.mel. They generate a command string according to your current preferences. An Example of a Smart Split would be:

eval(generateCmdString("-sp ",0));
(Click here to get to an information page which will give a more detailed explanation on dynamic Hotkeys)

This allows you to use the preferences to change the behavior of the mapped commands without having to change your hotkeys every time.
 Hotkeys - Use your mouse

Modern mouse devices usually have more than 3 mouse buttons. The additional mouse buttons can also be assigned some custom behavior. Pressing one of those is incredibly fast and saves you some of the valuable single hotkeys on your keyboard. This means, you can of course map something like Ctrl + Alt + SHIFT + x to your mouse button and invoke it with a single push.
In my case, I use the Microsoft Explorer 3.0, which allows me to invoke a smartSplit on the 4. button, and "delete edge" (the MEL Command for the hotkey is:
 polyPerformAction "polyDelEdge-cv true" e 0
on the 5. one while inside of Maya.

It is a real time saver and very convenient.

 Highspeed pointing with pen and tablet

Of course you can be fast with a mouse, but you will be faster with a tablet. Using a pen is more intuitive and more flexible as well.
Fortunately, Maya seems to be made to be controlled with a pen, so just give it a try.
But before you go buying a tablet, you should have the following things in mind:

1. It will take some time to get used to using a pen, and to navigate in the view port. The beginning will definitely be hard, but when you got it, you won't want to work with a ordinary mouse again.

2. When you've got only a small tablet, A5 for example, it may be hard to precisely select those tiny vertices.
In general, it will be better to use an A4 tablet.
If you were not happy with the precision even with a big tablet, you could increase your click box size (Window -- Settings/Preferences -- Preferences >> Selection -> Click Box Size)

 Highspeed Selection

During the modeling process it is essential to be able to quickly and easily select polygon components. Single picking them is only the last choice.
I made good experiences with Smart Edge Loop/Ring, Boundary and Paint Select put on hotkeys.

Especially the Paint Select Tool is vital because you can easily and intuitively make complex selections without having to concern about backfaces, which could easily be selected when using marquee select or the lasso tool.

Boundary turned out to be useful too in case of chamfer or solid Chamfer, so you would (paint) select the inner faces of the desired edge selection and hit the Boundary hotkey.
 UI - Expand your view

For modelers that already configured their hotkeys, all the other tiny bars like the shelve tend to be useless.
According to my own experience, the only thing necessary is a modelview and the channelbox to access the node's parameters.
The hotbox will give you access to all those functions that are in use very seldom and therefore don't have an own hotkey.

To get a modelingUI as shown in the picture above, simply deactivate every UI-Element except the channelbox.
To deactivate the menubar, you can press the default hotkey: 
[Ctrl] + [m]
OR execute the following command 
setMainMenubarVisible 0;
One thing necessary (which can normally be achieved by double-clicking on the Tool Icon which is not visible anymore) is to get the tools optionswindow. This may be necessary for the Paint Select Tool for example.

To achieve this, you should have a hotkey which executes the following command: 
toolPropertyWindow;
The hotkey will open the optionswindow for the tool currently active.

Please remember that you can easily change the space in which a translation or rotation or scaling occurs and other options by pressing and holding 
[w] or [e] or [r] + LeftMouseButton(hold)
 About your health

Now, you could consider yourself an unbeatable modeling machine, which could be true regarding the quality and speed of your work.
But unfortunately, this won't be true for your physical hull, also called body ;).

With today's screens, it might be very hard to damage your eyes, but with today's input devices and chairs, the hand's muscles could be damaged or a damage of the back could occur. These illnesses could easily keep you off your work for weeks, so you could loose your job in the worst case.

Keeping an eye on your health is essential to be able to stay healthy and productive, so I made up some guides which could help to stay healthy.
1. Try to work relaxed. When you stay in tension all the time, this 
may result in tendonitis.

2. Be active! When you sit most of the day in front of the screen, 
the brain does all the work, while your body is quite inactive. Try 
to keep a balance between brain and body activity and do some sports 
several times a week. You will definitely feel better when sports 
are a part of your life.
