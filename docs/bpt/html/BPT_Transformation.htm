Transformation


The Transformation Tools will help you to quickly rearrange your meshes.

The Smart Move Component command allows you to move components directly on the mesh, it feels a bit like the movement of isoParms on a NURBS surface. You can also make special selections that will move as if they were a Chamfer or a solid chamfer.
As opposed to Maya's native move component tool the edges polarity does not play a role, so the components will move as expected.

The Soft Transformation Tool allows you to move components, but as opposed to the native Move Tool, you can adjust a soft selection and influence the surrounding geometry.

Both tools work on vertices, so any other selection type will be implicitly converted to this component type.
To learn more about these tools, please go on reading.


 Smart Move Component

ByronsPolyTools -Flag ...
LongName ShortName Description
-smartMoveComponent -smc - allows you to move your selection with the parameters of the BPTNode
LongName ShortName --Additional Flags--
-slideNotLimited -snl - the slide parameter will work without its bounds (0 to 1)


 Philosophy

The Smart Move Component Command, which actually creates a ByronsPolyToolsNode, was created to help you rearranging your components quickly and efficiently.
In general, once the BPT Node is created, you can use all its parameters nearly exactly the same way as you are used to it by the other commands.
And you will be able to move your selection as if it was created by Smart Split, Solid Chamfer or Chamfer.
While sliding, the edges will move on the underlying faces which will keep the original shape of your mesh as good as possible.

 Smart Split - like Behavior



When you select a single line of edges, the command will make it behave as if it was created by a Smart Split. As mentioned above, you can use every parameter on the BPT Node the move the selection.


In this example, the slideRelative parameter is 1. In this case, the side parameter will flip the vectors along which the selected edges slide. This additionally helps to change the shape of the object.
Please Note: The logical behavior when flipping the sides would be, that the slide parameter flips its behavior( i.e. 0 will have the same position as formerly 1 and vice versa), which would be quite useless, so I decided to make it behave like that, at least when slideRelative is 1.

 Chamfer - like Behavior

When you select two stacked lines of edges, or a single line of faces, these components will behave as if they were created with the chamfer command.


 Solid Chamfer - like Behavior

When you select three stacked lines of edges, or a two stacked lines of faces, these components will behave as if they were created with the solid chamfer command.

 Complex Paths


With every one of the abilities mentioned above, you can also create complex paths, which may change directions.
