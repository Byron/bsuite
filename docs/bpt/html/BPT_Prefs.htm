Preferences


The Preferences allow you to save several parameters between the runs of Maya.

 General


Auto Switch Component Mask: Automatically switches to the edge component mask when you invoked an edge Ring/edge Loop or boundary with a different component type.

Slide without Bounds: When selected, the slide parameter will have no limits, which are normally 0 to 1

Slide is Relative: Will enable the "slideRelative" parameter when invoking a smart split.

Normal is Relative: Will enable the "normalRelative" parameter when invoking a smart split.

Boost Performance: If enabled, the performance of every deformation will be improved by 10% to 25% in shaded mode. This is possible by changing the meshes triangulation routine to a one which is optimized for speed. This routine resists in the "fastTrgNode1". There may only be one triangulation node of this type, otherwise BPT won't work as expected.
Please Note: When the routine has been set once, it will speed up every deformation, beginning with a simple move, ending with skin and complex deformers on the affected mesh, but it is not suited if there are any concave nGons in the mesh.

Important: To make sure the triangulation function will be found if you export your mesh, the "fastTrgNode1" will be connected to your mesh, so that it gets exported as well.

In case you duplicate your mesh, please make sure that "Duplicate Input Graph" is not selected, because it would duplicate the "fastTrgNode1" which will lead to errors with the MEL routine which drives BPT.

Please Note:
Because this triangulation function creates static triangles, this may lead to triangle overlaps in certain situations. To restore the native Maya triangulation, you just have to execute the following command:
setAttr meshShapeName.userTrg -type "string" "";
where meshShapeName is the name of your meshes shape which resides below its transform node.

Additionally, boost performance will be used automatically while inside the Soft Transformation Tool.


 Bevel

Connect Ends: Determines whether to connect the ends of the chamfer operation or leave them open creating nGons

Triangulate Ends: Determines whether to triangulate the connected ends or not

Use Slide of 0.2: If active, the slide will be set to 0.2 initially


 Initial Settings

Always Visualize Falloff: Starts Up the BPTNode with Falloff Visualization enabled

Always Visualize Original Mesh: Starts Up the BPTNode with Mesh Visualization enabled


 Falloff Options
In the Slide Falloff and Along Normal Falloff sub menus you can change which falloff type to use initially. As all the values in the preferences, these settings will be saved and therefore be consistent between different runs of Maya.

 STTool Settings

In this submenu, you can change the initial falloff for the Soft Transformation Tool.
The Always Visualize Falloff parameter will determine whether the weights should be visualized when entering the tool or not.
The Use Last Distance parameter will determine whether the last distance set should be used when you change selections.
Finally, the Normal is Relative switch will enable the normalIsRelative parameter on the softTransformationNode (which actually performs the deformation


 Additional Options

Update Preferences: Gather the option values of all sub menus and set the preferences accordingly

Propagate Preferences: The option values of the sub menus will be set according to your preferences


 Show Command
When you click on a menu item, a command string according to your preferences will be printed into the script editor. This shall assist you in creating your hotkey configuration.
See the Workflow for more information on hotkeys and possibilities to improve your workflow.
