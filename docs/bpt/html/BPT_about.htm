About ... ?


Byron's Poly Tools were created to assist me in modeling, and to make it more efficient. But I didn't want to create just a Plug-In Version of MJ PolyTools I wanted to go further, getting rid of the limitations imposed by the scripts, and adding new features that I have waited for a long time.

So I took all my MEL knowledge, threw it away (temporarily :) ), and started to learn Maya API. It was quite hard at the beginning (as everything is), but after a while, I got used to it :). One month was my personal deadline for learning Maya API and writing the Plug but ... after two months of development, the basic features were implemented, and working quite well :).

But then, I got new ideas, and started adding a working chamfer and solid chamfer, and soft selections, and a Soft Transformation Tool, and last but not least a smart move component ability. So, all in all, it took me 4 months of full-time work to learn Maya API and to write and debug the plugin. It definitely was worth it :).

This is the Version you have now, introducing ten tools:

   • Smart Split
   • Solid Chamfer
   • Chamfer
    • Smart Edge Loop
   • Smart Edge Ring
   • Boundary
   • Grow
   • Shrink
   • Smart Move Component
   • Soft Transformation Tool

Last but not least, I really suggest you to read through the whole documentation carefully. This will be necessary to learn about the "smart" features as well as about the limitations
