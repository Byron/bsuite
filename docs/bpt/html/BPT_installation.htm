Installation

 Extracting Files ...
  1. Use Winzip or any popular utility to extract the ByronsPolyToolsV1.0MayaX.zip
    archive to a temporary directory on your workstation.
  2. In the temporary directory you should see the following directory structure.


    3. /plugins
    /scripts
    disclaimer.txt
    installationinstructions.txt

  3. Copy the three files from the /scripts folder to your current maya user directory.
    Windows:
    /documents and settings/<User name>/my documents/maya/scripts
  4. Copy the primary plugin file ByronsPolyTools.mll from the plugin folder to the following directory.
    /program files/aliaswavefront/maya5.0/bin/plugins/
  5. Start Maya

There are two ways to install the Byrons Poly tools plugin into the Interface.

Installing the offline documentation

Please download the documentation's archive using one of the following links:

BPTDocumentation Link 1

or

BPTDocumentation Link 2

Once the download is completed, please extract the archive in a folder of your choice.

 Initalizing the Plug-In


Type "ByronsPolyToolsMenu" into the command line and hit enter.

OR

  1. Press Window ---->Settings/preferences--->Plugin Manager
  2. In the list of Maya plugin files your should see "ByronsPolyTools1.0"
    in the list. Check the Auto Load and Loaded box next to the entry which will
    load the plugin during the current and all subsequent sessions of Maya.
  3. You should see the Byrons Poly Tools 1.0 menu pop up in the top title bar if
    you are in modeling layout.


You should now be able to utilize all the tools available in the package.
